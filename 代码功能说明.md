# 代码功能说明

## 1. 总体流程

项目只依赖实时 `/scan` 激光话题作为核心输入：

```text
/scan
  -> LaserScan 转二维点
  -> scan-to-scan ICP 前端
  -> 无回环轨迹
  -> 关键帧
  -> 回环候选检测
  -> 回环 ICP 验证
  -> 位姿图优化
  -> 优化轨迹与结果 CSV
```

项目提供两个版本：

- 无回环基础版：`scan_icp_node.py`
- 回环 + 位姿图优化版：`pose_graph_slam_node.py`

所有结果统一保存在：

```text
catkin_ws/src/slamware_loop_slam/result
```

## 3. ROS 节点

### `scripts/scan_icp_node.py`

无回环基础版。功能：

- 订阅 `/scan`
- 过滤 LaserScan 无效距离
- 将 scan 转成二维点集
- 和上一帧 scan 做 ICP
- 累积位姿，发布 `/slamware_icp_pose` 和 `/slamware_icp_path`
- 保存 `result/no_loop_path.csv`
- 对 ICP 结果做质量检查，拒绝误匹配和大跳变

### `scripts/pose_graph_slam_node.py`

回环版主节点。功能：

- 复用 scan-to-scan ICP 作为前端里程计
- 按平移和旋转阈值选择关键帧
- 为关键帧生成轻量径向描述子
- 用局部位姿距离和描述子距离筛选回环候选
- 用 ICP 验证回环候选
- 添加里程计边和回环边
- 用 `scipy.optimize.least_squares` 做位姿图优化
- 发布局部轨迹和优化轨迹
- 保存局部轨迹、优化轨迹和回环边 CSV
- 额外保存 `pose_graph_keyframes.csv`，供后处理按关键帧 ID 正确绘制回环线

本版已经加入稳定性优化：未触发优化时不会把已有优化轨迹重置成局部轨迹；回环后的新关键帧会继承当前图优化修正。

## 4. 核心算法模块

### `src/slamware_loop_slam/scan_to_points.py`

把 `sensor_msgs/LaserScan` 转成二维点集。处理内容：

- `NaN` / `inf` 过滤
- 结合 ROS 消息自带 `range_min` / `range_max`
- 项目参数 `min_range` / `max_range`
- 按 `downsample_step` 降采样

### `src/slamware_loop_slam/icp.py`

2D ICP 主循环。处理内容：

- KDTree 最近邻匹配
- 最大对应距离门限 `max_correspondence_distance`
- 匹配裁剪 `trim_percentile`
- 最小匹配数 `min_correspondences`
- 可选质心初始化 `use_centroid_init`
- 返回旋转、平移、角度、变换后点云和误差历史

默认关闭质心初始化，更适合实时相邻 scan；因为实际雷达视野和遮挡变化时，质心初始化容易引入跳变。

### `src/slamware_loop_slam/align.py`

给定两组对应点，使用 SVD 估计二维刚体变换。

### `src/slamware_loop_slam/pose.py`

二维位姿工具：

- 角度归一化
- 位姿合成
- 位姿求逆
- 相对位姿
- 平面距离

### `src/slamware_loop_slam/loop_detection.py`

轻量回环候选检测：

- 每个关键帧生成径向直方图描述子
- 用描述子距离筛选外观相似帧
- 用局部位姿距离筛选空间邻近帧

### `src/slamware_loop_slam/pose_graph.py`

位姿图优化：

- 第一帧固定
- 每条边约束两个关键帧之间的相对位姿
- 里程计边和回环边使用不同权重
- 使用 `soft_l1` 鲁棒损失降低坏边影响

## 5. 关键参数

无回环参数文件：

```text
config/scan_icp_params.yaml
```

回环版参数文件：

```text
config/pose_graph_slam_params.yaml
```

重要参数：

- `max_correspondence_distance`：ICP 最近邻最大距离
- `trim_percentile`：保留较好匹配的比例
- `max_icp_error`：前端 ICP 最大允许误差
- `max_step_translation`：单帧最大平移跳变
- `max_step_rotation_deg`：单帧最大旋转跳变
- `max_consecutive_rejects`：连续拒绝多少帧后重置参考 scan
- `keyframe_translation_thresh`：关键帧平移阈值
- `keyframe_rotation_thresh_deg`：关键帧旋转阈值
- `loop_candidate_distance`：回环候选空间距离
- `loop_descriptor_threshold`：描述子距离阈值
- `loop_icp_error_threshold`：回环 ICP 误差阈值
- `loop_max_measurement_translation`：回环边最大平移
- `loop_max_measurement_rotation_deg`：回环边最大旋转
- `optimize_on_loop_only`：默认只在发现回环后做图优化，降低 Go1 上的 CPU 压力

## 6. 后处理

`tools/plot_results.py` 读取 result 里的 CSV 并生成：

- `no_loop_path.png`
